## Hello Gatsby
 
基本上，在我个人的使用体验中，完全可以吊打 Hexo，在完美拥有 Hexo 的各种体验和功能后之外，Gatsby 还拥有更丰富的自定义功能，并且基于 React 和各类插件，可以随心所欲的实现各种特效、功能。

**优点：**
1. 个性化设置更灵活；
3. 帮助你学习 Graphql；
4. 使用 React ，随心所欲的开发各种组件、样式；
5. 利用相关插件实现 MDX 功能，减少页面中 DEMO 外链；
6. 配合 GithubPage 可以快速部署静态页面；

> 注：除了 GithubPage，官网还提供了更多的部署方式。

## 快速搭建

初始化一个 Gatsby 博客非常简单，只需执一条条命令即可。

```js
$ npx gatsby new blog
```
